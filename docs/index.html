Ecosystem Service Web Service¶

Ecosystem Service Web Service (ESWS) is a method for creating interoperable analysis workflows with the Open Geospatial Consortium (OGC) Web Services (OWS) standards , especially the Table Join Service (TJS), Web Coverage Service (WCS), Web Feature Service (WFS), and Web Processing Service (WPS).

ESWS workflows are:

  • modular

  • redistributable

  • open
